## 🛠️ Project 1: File Integrity Monitor (FIM)
* **Objective:** Detect unauthorized tampering of critical files using secure cryptography.
* **Core Concept:** Developed a Python script that calculates the **SHA-256 cryptographic hash** of a sensitive configuration file. The program actively monitors the system and triggers an immediate critical security alert if a file's digital signature is altered by a threat actor.
* **Files:** See `fim.py` in this repository.

## 🌐 Project 2: Signature-Based Web Traffic Firewall
* **Objective:** Mitigate common OWASP Top 10 web application vulnerabilities.
* **Core Concept:** Built a rules-based traffic inspector using Regular Expressions (Regex) in Python. The script simulates a Web Application Firewall (WAF) analyzing raw web traffic inputs, successfully intercepting and neutralizing live **SQL Injection (SQLi)** and **Cross-Site Scripting (XSS)** attack variants.
* **Files:** See `waf_scanner.py` in this repository.
